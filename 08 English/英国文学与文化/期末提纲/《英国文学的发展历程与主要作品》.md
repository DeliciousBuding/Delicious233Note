

| Periods and Schools（时期与流派）                                  | major representatives（主要代表人物） | 代表作品（英文原名）                                                | 作品简介                                                                                              |
| ----------------------------------------------------------- | ----------------------------- | --------------------------------------------------------- | ------------------------------------------------------------------------------------------------- |
| 1. Middle Ages (395 - 1500)（中世纪（395 年 - 1500 年））            | Geoffrey Chaucer（杰弗里・乔叟）      | *The Canterbury Tales*（《坎特伯雷故事集》）                         | 以十音节双韵体写成，由总序和 24 个单篇故事组成，讲述一群去坎特伯雷朝圣的香客途中的奇异故事，描绘了中世纪英国社会千姿百态的生活风貌。                              |
| 2. Renaissance (14 - 16)（文艺复兴时期（14 - 16 世纪））                | Shakespeare（莎士比亚）             | *Romeo and Juliet*（《罗密欧与朱丽叶》）                             | 讲述维罗纳城蒙太古和凯普莱特两家儿女的爱情悲剧，他们因家族仇恨殉情，歌颂坚贞爱情，反对封建世仇和宗法势力。                                             |
|                                                             |                               | *The Merchant of Venice*（《威尼斯商人》）                         | 莎士比亚最具代表性的喜剧，通过鲍西亚选亲、杰西卡与罗兰佐恋爱和私奔及 “割一磅肉” 的契约纠纷三条线索展开，反映资本主义早期商业资产阶级与高利贷者的矛盾，塑造了夏洛克这一唯利是图的高利贷者形象。 |
|                                                             | F. Bacon（弗朗西斯・培根）             | *Of Study*（《谈读书》）                                         | 探讨读书的目的、方法和作用，文字简洁深刻，是培根哲学思想和文学风格的体现。                                                             |
|                                                             | Thomas More（托马斯・莫尔）           | *Utopia*（《乌托邦》）                                           | 描述了一个理想社会，批判当时现实社会的弊端，展现了作者对理想制度的设想。                                                              |
|                                                             | Edmund Spencer（埃德蒙・斯宾塞）       | *The Fairy Queen*（《仙后》）                                   | 长篇史诗，具有深刻的道德和政治寓意，对英国诗歌形式有重要影响。                                                                   |
| 3. Revolution (17)（革命时期（17 世纪））                             | John Milton（约翰・弥尔顿）           | *Paradise Lost*（《失乐园》）                                    | 取材于《圣经》中人类始祖被逐出伊甸园的故事，把撒旦塑造成反抗上帝的叛逆之神，体现追求自由的崇高精神，与《荷马史诗》《神曲》并称西方古典文学三大史诗。                        |
|                                                             | John Donne（约翰・多恩）             | *Songs and Sonnets*（《歌与短歌》）                               | 玄学派诗歌代表作，以独特意象和情感表达，探讨爱情、宗教等主题。                                                                   |
| 4. Enlightenment (L17th - 18th C.)（启蒙运动时期（17 世纪后期 - 18 世纪）） | Alexander Pope（亚历山大・蒲柏）       | *An Essay on Criticism*（《批评论》）                            | 探讨文学批评原则和方法，对英国文学批评发展有重要影响。                                                                       |
|                                                             | William Blake（威廉・布莱克）         | *Songs of Innocence*（《天真之歌》）*Songs of Experience*（《经验之歌》） | 通过对比展现人性和社会的不同方面，《天真之歌》充满纯真美好，《经验之歌》则揭示现实的苦难与复杂。                                                  |
|                                                             | Robert Burns（罗伯特・彭斯）          | *A Red, Red Rose*（《一朵红红的玫瑰》）                              | 苏格兰民谣风格诗歌，以热烈语言表达爱情，情感真挚动人。                                                                       |
| 5. Romanticism (1798 - 1832)（浪漫主义时期（1798 - 1832 年））         | William Wordsworth（威廉・华兹华斯）   | *Lyrical Ballads*（《抒情歌谣集》）                                | 与柯勒律治合著，标志英国浪漫主义文学开端，强调自然与内心情感，语言清新质朴。                                                            |
|                                                             | Shelley（雪莱）                   | *Ode to the West Wind*（《西风颂》）                             | 诗歌充满对自由、革命的向往，“冬天来了，春天还会远吗” 体现对未来的乐观信念。                                                           |
|                                                             | John Keats（约翰・济慈）             | *Ode on a Grecian Urn*（《希腊古瓮颂》）                           | 对古瓮上图案展开想象，思考艺术与美的永恒，语言优美，意境深远。                                                                   |
|                                                             | L. Byron（拜伦）                  | *Don Juan*（《唐璜》）                                          | 通过主人公唐璜的游历经历，讽刺当时欧洲社会的虚伪、腐败，展现广阔社会图景。                                                             |
| 6. Critical Realism（批判现实主义）                                 | Charles Dickens（查尔斯・狄更斯）      | *A Tale of Two Cities*（《双城记》）                             | 以法国大革命为背景，通过梅尼特医生的经历和贵族迫害农民的悲剧，展示英、法社会现实，表明革命必然性。                                                 |
|                                                             | Bronte sisters（勃朗特姐妹）         | *Jane Eyre*（《简・爱》，夏洛蒂・勃朗特）                                | 讲述孤女简・爱在磨难中追求自由尊严，最终获得幸福，塑造了敢于反抗、追求平等的女性形象。                                                       |
|                                                             | Robert Browning（罗伯特・勃朗宁）      | *My Last Duchess*（《我的前公爵夫人》）                              | 采用戏剧性独白，通过公爵对前公爵夫人画像的描述，揭示其狭隘自私的性格。                                                               |
|                                                             | Lord Tennyson（丁尼生勋爵）          | *Idylls of the King*（《国王之歌》）                              | 以亚瑟王传说为题材，探讨骑士精神、爱情、责任等主题，语言华丽，富有音乐性。                                                             |
| 7. Modernism (20th century)（现代主义（20 世纪））                    | Thomas Hardy（托马斯・哈代）          | *Tess of the d’Urbervilles*（《德伯家的苔丝》）                     | 讲述苔丝的悲剧命运，揭露社会道德对人性的压迫，反映 19 世纪末英国农村的社会问题。                                                        |
|                                                             | Oscar Wilde（奥斯卡・王尔德）          | *The Picture of Dorian Gray*（《道林・格雷的画像》）                  | 探讨美与道德的关系，道林・格雷的画像随其恶行而变化，揭示虚荣与罪恶的后果。                                                             |
|                                                             | Bernard Shaw（萧伯纳）             | *Saint Joan*（《圣女贞德》）                                      | 塑造了法国民族英雄圣女贞德的形象，探讨信仰、权力与人性的冲突。                                                                   |
|                                                             | D. H. Lawrence（D. H. 劳伦斯）     | *Sons and Lovers*（《儿子与情人》）                                | 分析家庭关系与情感纠葛，通过保罗与母亲、恋人的关系，展现人性的复杂与挣扎。                                                             |
|                                                             | James Joyce（詹姆斯・乔伊斯）          | *Ulysses*（《尤利西斯》）                                         | 意识流小说经典，描述 1904 年 6 月 16 日都柏林人物的生活经历，展现人类潜意识与精神世界。                                                |
| 8. Contemporary (1945 - present)（当代（1945 年至今））              | J. K. Rowling（J. K. 罗琳）       | *Harry Potter and the Philosopher’s Stone*（《哈利・波特与魔法石》）   | 讲述哈利・波特进入霍格沃茨魔法学校的冒险，充满奇幻元素，传递勇气、友谊与爱的主题。                                                         |
