# 初识算法

## 什么是算法？

算法是解决特定问题的一系列步骤。它可以是简单的数学计算，也可以是复杂的逻辑操作。

## 算法的特性

- **有穷性**：算法必须在有限的步骤内完成。
- **确定性**：算法的每一步都必须有明确的定义。
- **输入**：算法有零个或多个输入。
- **输出**：算法有一个或多个输出。
- **可行性**：算法中的每一步都必须是可行的，即能够通过有限次的基本操作实现。

## 常见算法类型

- **排序算法**：如快速排序、归并排序等。
- **搜索算法**：如二分查找、深度优先搜索等。
- **[[图论]]算法**：如最短路径算法、最小生成树算法等。
